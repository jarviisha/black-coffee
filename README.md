## 🚀 Project Status: Incubation Stage

The **Black Coffee** project is currently in its early development (incubation) phase. It is a modern social networking platform focused on seamless user experience and high performance.

---

## 🛠️ Tech Stack

The project is built using the latest industry-standard technologies:

- **Frontend:** [React 19](https://react.dev/), [TypeScript](https://www.typescriptlang.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **State Management:** [Zustand](https://github.com/pmndrs/zustand)
- **Data Fetching:** [TanStack Query v5](https://tanstack.com/query/latest)
- **Routing:** [React Router v7](https://reactrouter.com/)
- **Form Handling:** [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)
- **API Client Generation:** [Kubb](https://kubb.dev/) (from OpenAPI)
- **Deployment:** [Cloudflare Wrangler](https://developers.cloudflare.com/workers/wrangler/)
- **Testing:** [Vitest](https://vitest.dev/)

## 📂 Project Structure

```text
src/
├── api/          # API hooks and clients auto-generated by Kubb
├── app/          # Providers and Router configuration
├── features/     # Feature-based modules (auth, feed, post, profile...)
├── components/   # Shared UI components
├── store/        # Global state management (Zustand)
├── hooks/        # Shared custom hooks
├── lib/          # Library configurations (axios, i18n...)
└── styles/       # CSS and themes configuration
```

## 🏗️ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (LTS version)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone <your-repo-url>
   cd black-coffee
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

### Available Scripts

- `npm run build`: Build the application for production.
- `npm run lint`: Run ESLint to check for code issues.
- `npm run test`: Execute unit tests with Vitest.
- `npm run format`: Format code using Prettier.
- `npm run deploy`: Deploy the application to Cloudflare.
